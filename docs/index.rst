Clastic Documentation
=====================

Clastic is the simple Python web framework that looks out for **you**.

It features:

* Familiar, but cleaner-cut building blocks
  * Applications
  * Routes
  * Middlewares
  * Templates and Renderers
* Simple deployment with native WSGI
* Full introspectability
* No global state or import-time side effects
* A general lack of bloat, cruft, and pits
